Comunicação Client-Server

A comunicação cliente-servidor é um modelo de troca de dados em que dois lados têm papéis definidos:

  • Cliente: O Cliente faz as requisições, que Podem ser um acesso ao navegador, aplicativo ou dispositivo que quer acessar, ou até um ou informação.
  • Servidor: O Servidor responde às requisições do cliente, geralmente entregando dados, realizando tarefas ou guardando informações.

Características Principais:

  • Assincronismo: o cliente e o servidor podem funcionar em máquinas diferentes.
  • Escalabilidade: vários clientes podem se conectar ao mesmo servidor sem problemas.
  • Modularidade: o cliente não precisa saber como o servidor faz as coisas, só que ele responde corretamente.

Estrutura HTML

  • !DOCTYPE: Define o tipo do documento, nesse caso (HTML5).
  • HTML: Contém o conteúdo da página.
  • Head: Contém informações como titulo, meta, tags e links de estilo.
  • Title: Define o título que aparece na aba do navegador.
  • Body: É onde está o conteúdo visível ao usuário ex:(textos, imagens).
  • h1/h6: São tags usadas para cabeçalhos e Títulos.
  • p: Tag usada para paragrafos.
  • a: Tag usada para links de outras páginas ou sites.

Formulários

Formulários permitem coletar dados dos usuários, como nome, e-mail, senha, preferências, mensagens, entre outros. Eles são essenciais em cadastros, pesquisas, login e contato.

Os Tipos mais comuns de inputs, usados no formulário são:

  • type="password" - cria um campo para senhas
  • type="radio" - cria uma caixa de seleção, de uma unica seleção possivel.
  • type="checkbox" - cria uma caixa de seleção de multipla escolha.
  • type="file" - local para enviar arquivos.
  • type="date" - campo de seleção de Data.

Html Sêmantico

HTML semântico é uma forma de escrever código HTML utilizando tags que descrevem claramente o significado do conteúdo.

Principais tags:

  • Header - Cabeçalho da Página ou seção
  • Nav - Área de navegação (Ex. menu, links)
  • Main - Conteúdo principal da página
  • Section - Seção de conteúdo
  • Article - Seção para (posts, notícias, artigos)
  • Footer - Rodapé da página
  • Figure e Figcaption - usados para colocar legendas nas imagens
